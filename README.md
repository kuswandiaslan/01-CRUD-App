# CRUD App with Streamlit

simple CRUD Application (Create, Read, Update, Delete) with Streamlit and SQLite.

## Features
- Create (Create New Data)
- Read (View Data)
- Update (Edit Data)
- Delete (Delete Data)

## How to run Application
    pip install -r requirements.txt
    streamlit run app.py
